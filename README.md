# Claude Code Skills

Global skills for [Claude Code](https://claude.ai/claude-code).

## Structure

Each skill lives in its own directory with a `SKILL.md` entry point and optional `references/` and `scripts/` subdirectories.

## Installation

```bash
git clone https://github.com/BlakeHastings/skills.git ~/.claude/skills
```

## Skills

| Skill | Description |
|-------|-------------|
| `kroger-api` | Interact with the Kroger Public API (products, locations, cart, identity) |
| `manage-skills` | Create, edit, list, delete, and validate Claude Code skills |
